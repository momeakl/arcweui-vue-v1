#安装/使用

   npm install arcweui-vue-v1 --save

   main.js 引入依赖：
``` javascript
import ArcWeUIV1 from 'arcweui-vue-v1'
import 'arcweui-vue/dist/arcweui-vue-v1.min.css'
Vue.use(ArcWeUIV1)
```
   具体使用请到github 下载项目看examples
